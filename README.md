# ⛽ โครงการ "เช็คปั๊มพิจิตร" (Phichit Fuel)

ยินดีต้อนรับสู่โปรเจกต์ **"เช็คปั๊มพิจิตร" (Let's Help Phichit)** แพลตฟอร์มอาสาสมัคร (Crowdsourcing) ที่ออกแบบมาเพื่อให้ชาวพิจิตรสามารถตรวจสอบและแบ่งปันสถานะน้ำมันในปั๊มต่างๆ ทั่วจังหวัดพิจิตรได้แบบเรียลไทม์ เพื่อช่วยแก้ปัญหาความไม่แน่นอนของปริมาณน้ำมันในช่วงวิกฤตครับ!

---

## 🌟 จุดประสงค์โครงการ (Project Objective)
เพื่อสร้างฐานข้อมูลที่ขับเคลื่อนโดยชุมชน ช่วยให้ผู้ขับขี่รถยนต์ในพิจิตรไม่ต้องเผชิญกับปัญหาน้ำมันหมด หรือการรอคิวที่ยาวนานโดยไม่จำเป็น ผ่านการรายงานสถานะตรงจากผู้ใช้งานที่อยู่หน้าปั๊มจริงๆ

---

## ✨ คุณสมบัติเด่น (Key Features)
*   **📍 แผนที่ตรวจสอบ (Interactive Map)**: แสดงหมุดปั๊มน้ำมันตามสีสถานะ (เขียว = มี, เหลือง = รอนาน, แดง = หมด)
*   **📝 ระบบอาสาสมัคร (Contributor System)**: ผู้ใช้สามารถเข้าสู่ระบบผ่าน Google เพื่อรายงานสถานะน้ำมันที่ปั๊ม (ต้องอยู่ในระยะ 200 เมตร)
*   **📊 แดชบอร์ดข้อมูล (Data Dashboard)**: สรุปภาพรวมปั๊มทั้งหมด และรายการปั๊มที่ใกล้ตำแหน่งปัจจุบันที่สุด
*   **📱 ดีไซน์ทันสมัย (Modern UI)**: ใช้หน้าตาแบบ **Opaque Pill Dock** (แถบเมรูทรงแคปซูล) และฟอนต์ **Prompt** ที่อ่านง่ายสำหรับคนทุกวัย
*   **🧭 ระบบนำทาง (Navigation)**: เชื่อมต่อโดยตรงกับ Google Maps เพื่อนำทางไปยังปั๊มที่ต้องการ

---

## 🛠️ เทคโนโลยีที่ใช้ (Technology Stack)
*   **หน้าบ้าน (Frontend)**: HTML5, CSS3, JavaScript (Vanilla JS)
*   **ระบบแผนที่ (Mapping)**: Leaflet.js ร่วมกับ Google Maps Tiles
*   **ฐานข้อมูล (Database)**: Firebase Firestore (Real-time updates)
*   **การยืนยันตัวตน (Authentication)**: Google Identity Services / Firebase Auth
*   **หลังบ้าน (Backend API)**: Node.js (Deployed on Render)

## 🔐 ความปลอดภัยและความเป็นส่วนตัว (Privacy)
*   ระบบเก็บข้อมูลเฉพาะชื่อและอีเมลจากการล็อกอิน Google เพื่อใช้สำหรับสถิติน้ำใจและการเข้าถึงสิทธิ์การรายงานเท่านั้น
*   การระบุพิกัด GPS จะทำเฉพาะในเบราว์เซอร์ของผู้ใช้ และจะส่งไปยังเซิร์ฟเวอร์เฉพาะตอนกดยืนยันรายงานเพื่อพิสูจน์ระยะทาง 200 เมตรเท่านั้น

---

## 🧡 ร่วมเป็นส่วนหนึ่ง
คุณสามารถร่วมพัฒนาโครงการนี้ได้โดยการช่วยทดสอบการใช้งาน หรือการแชร์ลิงก์ให้คนในจังหวัดพิจิตรได้ใช้งานกันอย่างแพร่หลาย ยิ่งมีคนใช้มาก ข้อมูลก็จะยิ่งแม่นยำครับ!
